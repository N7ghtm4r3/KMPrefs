    Overview

    Kotlin Multiplatform Pref(erence)s system allows you to locally store, retrieve, and remove data on each platform, leveraging the native APIs provided by each platform:

    • The Android target leverages the SharedPreferences APIs
    • The iOS and macOS targets leverage the UserDefaults APIs
    • The JVM target leverages the Preferences APIs
    • The Web target leverages the LocalStorage APIs

    Implementation
